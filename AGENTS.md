# AGENTS.md

이 프로젝트는 Cafe24 스마트디자인 템플릿입니다.

## 작업 기준

- Cafe24 스마트디자인 문법과 런타임 규칙을 보존합니다.
- `$변수`, `{$변수}`, `<!--@layout-->`, `<!--@css-->`, `<!--@js-->`, `<!--@import-->` 같은 템플릿 지시문을 임의로 바꾸지 않습니다.
- `module="..."`, `ec-data-*`, `data-ez-*`, `ez-*` 속성은 Cafe24 편집기와 모듈 동작에 연결될 수 있으므로 삭제하거나 이름을 바꾸지 않습니다.
- HTML 구조를 정리할 때도 Cafe24 모듈의 반복 영역, 조건 영역, 상품/회원/주문 관련 바인딩을 유지합니다.
- 기존 폴더 구조와 파일 위치를 우선 따릅니다.

## 수정 시 주의사항

- 디자인 수정은 관련 HTML, CSS, JS 파일에 한정해 최소 범위로 처리합니다.
- Cafe24 관리자 또는 스마트디자인 편집기에서 생성한 것으로 보이는 주석, 속성, 래퍼 요소는 특별한 이유 없이 제거하지 않습니다.
- 외부 라이브러리를 추가하기 전에 기존 `css/`, `js/`, `layout/`, `ez/` 구조에서 이미 쓰는 방식이 있는지 먼저 확인합니다.
- 인코딩과 줄바꿈은 기존 파일 스타일을 최대한 유지합니다.
- 템플릿 변수를 일반 정적 문자열처럼 취급하지 않습니다.

## 검증

- 변경 후에는 깨진 태그, 누락된 닫힘 태그, 잘못된 Cafe24 지시문이 없는지 확인합니다.
- 가능하면 브라우저에서 주요 화면이 정상 렌더링되는지 확인합니다.
- Cafe24 전용 데이터는 로컬에서 완전히 재현되지 않을 수 있으므로, 로컬 확인 결과와 실제 Cafe24 적용 결과가 다를 수 있음을 염두에 둡니다.
